# Branchen-Templates für Web-Agenturen

Hochwertige, responsive HTML-Templates für lokale Handwerksbetriebe und Dienstleister.

## Live-Demos

- **Elektriker:** [elektriker/](https://martin-scharf.github.io/branchen-templates/elektriker/)
- **Tischler:** [tischler/](https://martin-scharf.github.io/branchen-templates/tischler/)
- **SHK (Sanitär/Heizung/Klima):** [shk/](https://martin-scharf.github.io/branchen-templates/shk/)
- **Maler:** [maler/](https://martin-scharf.github.io/branchen-templates/maler/)
- **Friseur:** [friseur/](https://martin-scharf.github.io/branchen-templates/friseur/)

## Features

- Mobile-First, vollständig responsive
- Professionelles, hochwertiges Design
- Keine externen Abhängigkeiten (nur Google Fonts)
- Impressum & Datenschutz Links vorbereitet
- Schnelle Ladezeiten durch Inline-CSS
- Semantisches HTML5
- Barrierefreie Navigation

## Verwendung

Jedes Template ist eine einzelne HTML-Datei mit integriertem CSS. Einfach:

1. Template-Ordner kopieren
2. Texte und Kontaktdaten anpassen
3. Bilder einfügen (Platzhalter sind vorbereitet)
4. Google Maps einbetten
5. Impressum & Datenschutz Seiten erstellen
